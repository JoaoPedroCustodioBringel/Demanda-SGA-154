# Task Manager
Sistema de gerenciamento de tarefas com Angular 20+ e NestJS.

## Tecnologias
- **Frontend**: Angular 20 + Bootstrap 5 + Signals + ZoneLess
- **Backend**: NestJS + TypeORM + SQLite

## Executar

### Backend
```bash
cd back-sg154
npm install
npm run start:dev
```
Servidor: `http://localhost:3000`

### Frontend
```bash
cd front-sga154
npm install
npm start
```
Aplicação: `http://localhost:4200`

## Funcionalidades
- CRUD completo de tarefas
- Dashboard com estatísticas
- Formulários reativos com validações
- Design responsivo
